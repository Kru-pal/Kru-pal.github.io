Automated US-VISA Appointment Checker

About

Welcome to the Automated US-VISA Appointment Checker! This tool automates the process of checking for available appointment dates for US-VISA applications, saving you time and effort.

Features

  • Automated browsing using Puppeteer
  • Real-time email notifications
  • Customizable configuration

Usage

To use the Automated US-VISA Appointment Checker, simply visit the GitHub Pages of the project and follow the instructions.
